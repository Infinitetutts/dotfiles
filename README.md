# dotfiles

### Tmux Requirements
* Tmux requires the following for battery to work in status line

**Install**   
`curl -OL https://raw.github.com/richo/battery/master/bin/battery`  
`mv battery ~/bin/battery`  
`chmod +x ~/bin/battery`  

**Commands**  
`~/bin/battery Charging`  
`~/bin/battery Discharging`
